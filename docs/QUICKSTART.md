# Quickstart Guide

Get up and running with Smart Heating in minutes.

## 1. Install the Integration
- Use HACS (recommended):
  - Add custom repo: `https://github.com/TheFlexican/smart_heating`
  - Search for "Smart Heating" and install
  - Restart Home Assistant
- Or manual install: copy to `/config/custom_components/smart_heating`

## 2. Configure in Home Assistant
- Go to Settings → Devices & Services → Add Integration
- Search for "Smart Heating" and follow the wizard
- Assign climate devices to Home Assistant areas

## 3. Open the Web Interface
- Use the sidebar link in Home Assistant
- View and control all areas, devices, and schedules

## 4. Set Up Schedules & Modes
- Configure time-based, preset, or date-specific schedules per area
- Use boost and vacation modes for comfort and savings

## 5. Add Sensors & Safety
- Integrate window, presence, smoke, CO, and gas sensors for automation and safety

## 6. Explore Analytics & History
- View temperature history, efficiency, and recommendations

## 7. Backup & Restore
- Export/import configuration, migrate between JSON and database storage
